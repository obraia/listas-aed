Esse programa foi desenvolvido para ser testado por linha de comando.
Para testar o programa basta:
1. Compilar com javac AppArgs.java
2. Executar o programa com: java AppArgs quantidadeRegistros ordemEntrada OrdemSaida
Ex: java AppArgs 2000 decrescente crescente
3. Esperar os testes terminarem, e vericiar os resultados na pasta ./relatorios
